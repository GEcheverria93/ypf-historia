# YPF – 103 años de historia

Sitio estático (HTML/CSS/JS) que presenta una línea de tiempo histórica sobre YPF.

Cómo desarrollarlo localmente:

1. Instala dependencias si las necesitas (opcional):

```pwsh
npm install
```

2. Usar Live Server (recomendado en VS Code):
- Instala la extensión Live Server en VS Code y haz “Go Live” sobre `index.html`.

O con `npx live-server`:

```pwsh
cd 'c:\Users\PC\Desktop\dev\censYPF'
npx live-server . --port=5500
```

Cómo subir a GitHub y desplegar en Vercel (resumen):

1. Inicializar git y hacer un commit
2. Crear repositorio en GitHub y empujar
3. Conectar Vercel a GitHub y crear un proyecto o usar vercel CLI

Referencias: Vercel - https://vercel.com/docs
